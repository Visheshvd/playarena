üì° Available API Endpoints

Authentication

POST /api/auth/send-otp
Send OTP to mobile number
POST /api/auth/verify-otp
Verify OTP and login
GET /api/auth/me
Get current user (requires auth)

Pricing

GET /api/pricing
Get all pricing information

Bookings

POST /api/booking/create
Create a new booking (requires auth)
GET /api/booking/available-slots
Get available time slots
GET /api/booking/my-bookings
Get user's bookings (requires auth)

Matches

GET /api/matches/ongoing
Get currently ongoing matches
GET /api/matches/past
Get past matches (requires auth)
GET /api/matches/records
Get match statistics (requires auth)
